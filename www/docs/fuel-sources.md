# Источники данных о топливе (moto-hud)

Сводка для разработки и юридической оценки интеграций.

## Текущая схема в приложении

| Слой | Источник | Что даёт |
|------|----------|----------|
| Геометрия АЗС | [OpenStreetMap Overpass](https://wiki.openstreetmap.org/wiki/Overpass_API) | координаты, бренд, `osm_id` |
| Наличие | [ГдеБЕНЗ](https://gdebenz.ru) `/api/nearby` | `yes` / `queue` / `low` / `no`, подтверждения |
| Краудсорс | `localStorage` (`moto-hud-fuel-crowd-v1`) | отметки водителя в HUD, приоритет 6 ч |
| Прокси (веб) | Cloudflare Worker `workers/gdebenz-proxy` | обход CORS для PWA |

Порядок загрузки наличия (веб): **прокси** → прямой запрос → corsproxy.io → allorigins.

## ГдеБЕНЗ

- **API:** `GET https://gdebenz.ru/api/nearby?lat=&lon=&radius_km=`
- **CORS:** нет для браузера; в APK — `CapacitorHttp`
- **Сопоставление:** `osm_id`, иначе ближайшая точка &lt; 100 м
- **Прокси:** `workers/gdebenz-proxy` — деплой `npx wrangler deploy`, URL в ⚙ **URL прокси ГдеБЕНЗ**
- **Ограничения:** периодические 503 с серверов CI; кэш 5 мин в `sessionStorage`

## ГдеЗаправка (gdeto-zapravka.ru)

- **Публичного документированного API нет**
- Spike: `npm run fuel:spike` (`scripts/fuel-gdzapravka-spike.mjs`) — перебор типовых путей
- **Рекомендация:** партнёрское соглашение / white-label feed, не скрейпинг карты
- Альтернатива для продукта: оставить OSM + ГдеБЕНЗ + краудсорс

## Другие сервисы (не подключены)

| Сервис | Комментарий |
|--------|-------------|
| [Benzuber](https://benzuber.ru) | B2B, топливные карты, не крауд-наличие на карте |
| [Бензап](https://benzap.ru) | цены/акции, нет открытого nearby API |
| Т-Банк / приложения банков | закрытые API, только для своих клиентов |
| Яндекс Заправки | закрытая экосистема |

## Краудсорс в HUD

- Кнопки **есть / очередь / мало / нет** в панели ⛽ (режим 1–2)
- Привязка к ближайшей АЗС в радиусе 200 м или к выбранной
- Хранение только на устройстве; экспорт JSON — `exportCrowdReportsJson()` (для будущей синхронизации)
- Свежая отметка (&lt; 6 ч) перекрывает статус ГдеБЕНЗ

## Приватность

- Координаты уходят на Overpass и ГдеБЕНЗ (или ваш Worker) при поиске АЗС
- Краудсорс не отправляется на сервер без отдельной функции «поделиться»
- См. также `docs/privacy.html`

## Деплой прокси (кратко)

```bash
cd workers/gdebenz-proxy
npx wrangler login
npx wrangler deploy
```

В moto-hud: ⚙ → вставить `https://….workers.dev` → перезапустить поиск ⛽.

Опционально: маршрут `yourdomain.com/api/fuel/*` на Worker — тогда поле прокси можно оставить пустым (same-origin).
