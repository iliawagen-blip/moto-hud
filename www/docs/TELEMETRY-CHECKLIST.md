# Чек-лист телеметрии (полевые тесты)

## Функциональность

- [ ] Телеметрия **выключена** по умолчанию; `telemetry.log` — no-op
- [ ] Включение: тумблер в опциях или `?telemetry=1`
- [ ] Кнопка **📍 метка** видна только при активной сессии (HUD + включённая телеметрия)
- [ ] Одинарный тап — `mark`; двойной — `note: critical`
- [ ] Сессия ≥ 10 мин: `perf.avg_ms` начала ≈ конца (нет деградации rAF)
- [ ] Экспорт JSONL открывается replay в sim.html
- [ ] Метки (красные риски) на таймлайне
- [ ] Принудительное закрытие вкладки → сессия `dirty=true` при следующем запуске

## Типы событий

- [ ] `fix`, `snap`, `nav`, `sys`, `perf`, `audio`, `mark`, `meta`

## Пропуски (нет точки вставки)

- `sys.wakelock_lost` — только при `release` Screen Wake Lock API (PWA); нативный KeepAwake release не логируется отдельно
