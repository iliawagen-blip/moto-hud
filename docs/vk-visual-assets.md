## 2. Аватар сообщества

**Сейчас:** `moto-ils-vk-avatar.png` или telegram-avatar — что лучше читается в круге 200×200.

**Цель (когда будет время):** не абстрактная иконка, а кроп реального HUD из sim:

- квадрат от 400×400 (лучше 800×800);
- крупная стрелка манёвра + кусок скорости;
- тёмный фон, высокий контраст;
- без мелкого текста и без слова «бета» на аватаре;
- в круглой обрезке VK стрелка не должна обрезаться пополам — держать якорь в центре.

## 3. Обложка 1920×768

**Сейчас:** оставить `moto-ils-vk-cover.png`.

**Инварианты VK:**

- важный текст только в **центральной горизонтальной полосе** (верх/низ режутся на мобиле);
- «МОТО ИЛС» читается на ширине телефона;
- не плодить мелкие иконки по краям.

**Если переделывать позже:** тот же POV «руль + дорога», но стрелка/скорость совпадают с актуальным UI из sim (чтобы не врать интерфейсом 2025-го на обложке 2026-го).

## 4. Скриншоты для запуска (sim) — снять / отобрать

Минимум **5–7 кадров**. Все — вертикаль телефона (как в пресете sim) или кроп под VK.

| # | Сцена | Что в кадре | Для какого поста / слота |
|---|--------|-------------|---------------------------|
| S1 | Трасса HUD | Скорость ~110–120, стрелка манёвра, дистанция ~300–400 м, без меню | Пост 1–2, календарь A |
| S2 | Город / развязка | Карта overview с петлёй или HUD с близким манёвром (~80–150 м) | Пост 3 |
| S3 | Камеры | Онбординг «только в спину?» или HUD с предупреждением камеры | Пост 4 |
| S4 | Импорт | Поле финиша / баннер импорта Яндекс (если воспроизводится) | Пост 5 |
| S5 | Тема day | Высокий контраст «как в солнце» | Календарь нед.1 |
| S6 | Тема night / vintage | Тёмный HUD, читаемые цифры | Календарь нед.2 |
| S7 | Бета-карточка | S1 + крупная надпись на втором слое: «Бета · отзыв = доступ навсегда» | Пост 7 |
| S8 | Demo portrait GIF | `hud-demo-portrait-serpentine.gif` | Пост 1 (закреп) |
| S8b | Demo landscape GIF | `hud-demo-landscape-themes.gif` | Пост 3 / нед.1 |
| S9 | Топливо ⛽ | `fuel-assistant.png` или кадр из portrait GIF | Пост 10 |

### Пересъём demo-GIF (рекомендуется)

```bash
npm run build
npm run vk:gifs                 # оба
npm run vk:gifs:portrait        # только 9:16
npm run vk:gifs:landscape       # только 16:9
```

Нужны сеть (OSRM). Кадры во временных `_frames-*` (не коммитить).  
Соотношения: portrait **360×640 (9:16)**, landscape **720×405 (16:9)** — без растягивания, cover-crop.

### Как снять в sim за один вечер

1. Открыть `sim.html`, выбрать пресет телефона (тот же, что в закрепе).  
2. Загрузить маршрут / GPX / финиш с трассовым куском.  
3. ▶ ПОЕХАЛИ — дождаться движения.  
4. Пауза / скрин в момент крупной стрелки (не на 0 км/ч).  
5. Сменить тему → ещё 1–2 кадра.  
6. Отдельно открыть онбординг камер для S3.  
7. Экспорт без рамок браузера (только iframe телефона) — чище для ленты.

**Не публиковать:** экраны с 0 км/ч и пустой стрелкой как «главный» креатив — выглядят мёртвыми.
