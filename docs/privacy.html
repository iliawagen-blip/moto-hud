1. Общие положения

1.1. Настоящая Политика определяет порядок обработки и защиты информации при использовании веб-приложения и/или мобильного приложения «Мото ИЛС» (далее — ПО).

1.3. Используя ПО, вы подтверждаете ознакомление с настоящей Политикой. При несогласии — прекратите использование ПО.

2. Какие данные обрабатываются

2.1. Локально на устройстве (не передаются Оператору автоматически)

  • координаты GPS, скорость, курс, точность позиционирования;
  • построенные маршруты, избранные места, настройки интерфейса;
  • кеш последней поездки (маршрут, камеры) в localStorage;
  • записи телеметрии тестовых поездок в IndexedDB — только если вы включили телеметрию в настройках;
  • факт согласия с отказом от ответственности (версия и дата) в localStorage.

2.2. Данные, которые вы передаёте Оператору добровольно

  • адрес электронной почты на странице поддержки проекта — для формирования кассового чека при уплате вознаграждения (email сохраняется в sessionStorage браузера до отправки на сервер; при ручном режиме чека Оператор может запросить email отдельно);
  • содержание обращений на iliawagen@gmail.com;
  • обезличенные данные телеметрии — только при включённой opt-in функции и только после явной загрузки на сервер Оператора (когда функция будет доступна).

2.3. Данные, которые ПО не собирает

  • ФИО, паспортные данные, номер телефона — за исключением случаев, когда вы сами указываете их в письме или при оплате через банк;
  • контакты, фото, файлы устройства — без вашего явного действия (импорт избранного, выбор GPX и т.п.).

3. Сторонние сервисы

Для работы навигации ПО обращается напрямую с вашего устройства к публичным сервисам:

  • OpenStreetMap — карточные тайлы (лицензия ODbL);
  • OSRM — построение маршрута (лицензия BSD-2-Clause);
  • Overpass API — данные о камерах и POI;
  • Nominatim — поиск адресов;
  • ГдеБЕНЗ (gdebenz.ru) — опционально, статус наличия топлива на АЗС (только при нажатии ⛽ в нативном приложении).

Оператор не контролирует политику конфиденциальности указанных сервисов. При обращении к ним ваш IP-адрес и запрос могут обрабатываться их операторами. Рекомендуется ознакомиться с условиями соответствующих проектов.

4. Цели обработки

  • отображение навигационной информации на устройстве пользователя;
  • формирование кассового чека при добровольной оплате по публичной оферте;
  • ответ на обращения пользователя;
  • улучшение качества ПО на основании обезличенной телеметрии (только opt-in).

5. Правовые основания

Обработка персональных данных осуществляется в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных» на основании:

  • согласия субъекта персональных данных (телеметрия, email для чека, обращения);
  • исполнения договора (оферта) при уплате вознаграждения;
  • законных интересов Оператора при ответе на обращения.

6. Хранение и защита

6.1. Данные на устройстве хранятся до их удаления пользователем (очистка данных приложения/браузера) или переустановки ПО.

6.2. Email для чека в sessionStorage — до закрытия вкладки/сессии браузера.

6.3. Данные обращений и платежей на стороне Оператора хранятся в сроки, установленные законодательством РФ (бухгалтерский и налоговый учёт).

6.4. Оператор принимает разумные организационные меры защиты данных, передаваемых по email и через серверные системы.
