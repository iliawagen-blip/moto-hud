# OEM-матрица: фоновый GPS (Фаза 2)

Чек-лист полевых тестов foreground-service навигации на Android.
Цель: **0 разрывов GPS > 5 с** за 2 ч поездки на каждом устройстве.

## Минимальный набор (≥ 3 вендора)

| Вендор | Модель (пример) | Android | Прошивка | Батарея без огран.autostart | 2 ч HUD, экран off | 2 ч HUD, свёрнуто | Заметки |
|--------|-----------------|---------|----------|-----------------------------|--------------------|--------------------|---------|
| Samsung | Galaxy A / S | 13+ | One UI | ☐ | ☐ | ☐ | |
| Xiaomi | Redmi Note | 13+ | MIUI / HyperOS | ☐ | ☐ | ☐ | Autostart + «Без ограничений» |
| Huawei | P / Honor (HMS) | 10+ | EMUI / Harmony | ☐ | ☐ | ☐ | Protected apps |
| OnePlus | Nord / число | 13+ | OxygenOS | ☐ | ☐ | ☐ | |
| Google | Pixel | 14+ | stock | ☐ | ☐ | ☐ | эталон |

## Сценарий теста

1. Установить сборку из Play Console (closed testing) или `adb install`.
2. Построить маршрут ≥ 30 км, «ПОЕХАЛИ», уведомление «Навигация активна» видно.
3. **A:** экран погашен 30 мин; **B:** приложение свёрнуто 30 мин; **C:** повтор 2 ч суммарно.
4. Логировать: разрывы GPS > 5 с, убийство процесса, пропажа уведомления FGS.
5. Заполнить таблицу; при FAIL — скрин настроек батареи + модель прошивки.

## Настройки пользователя (онбординг в приложении)

- Отключить оптимизацию батареи для «Мото ИЛС».
- Разрешить autostart / фоновую активность (MIUI, EMUI, ColorOS).
- Не закрывать из «последних» во время навигации.
- Wake-lock: `@capacitor-community/keep-awake` при активном HUD.

## Критерий Фазы 2

GPS стабилен на **≥ 3 вендорах** из таблицы; известные OEM-workaround задокументированы в «Как пользоваться».
