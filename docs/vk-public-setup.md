```
🏍 Мото ИЛС — мотонавигатор в формате HUD для поездок на мотоцикле.
```

### Пост 2 — Как открыть и установить

```
📱 Как пользоваться

Открывается в браузере на телефоне, компьютере или планшете. Скачивать из магазина не нужно — это сайт с установкой на главный экран.

🤖 Android (Chrome / Яндекс.Браузер):
⋮ → «Добавить на главный экран» или «Установить приложение»
При первом запуске — разрешите геолокацию и звук.

🍎 iPhone / iPad (только Safari!):
«Поделиться» → «На экран Домой»
В Chrome на iOS установка не работает.

💡 Перед поездкой: закрепите телефон на руле, подключите Bluetooth-гарнитуру шлема, проверьте зарядку. Голосовые подсказки — основной канал информации в движении.

Подробные инструкции — в следующих постах и в комментариях к этому.
```

### Пост 3 — Android подробно

```
🤖 Установка на Android — пошагово

1. Откройте в Chrome или Яндекс.Браузере:
https://iliawagen-blip.github.io/moto-hud/

2. Меню (⋮) → «Добавить на главный экран» / «Установить приложение»

3. При первом запуске:
   • геолокация — обязательно
   • звук — для голосовых подсказок

4. В настройках телефона отключите автогашение экрана на время поездки (или включите «не гасить экран» в настройках приложения, если есть).

Приложение пытается удержать экран через WakeLock; на некоторых прошивках нужны дополнительные разрешения.

🎮 Сначала попробуйте симулятор:
https://iliawagen-blip.github.io/moto-hud/sim.html
```

---

### Пост 4 — iOS подробно

```
🍎 Установка на iPhone и iPad

⚠️ Только Safari — в Chrome на iOS PWA не ставится.

1. Откройте в Safari:
https://iliawagen-blip.github.io/moto-hud/

2. Кнопка «Поделиться» (квадрат со стрелкой) → «На экран Домой»

3. Разрешите геолокацию: «При использовании приложения»

4. Для голоса — выключите беззвучный режим.

На iOS полноэкран и удержание экрана слабее, чем на Android. На длинной поездке иногда нужно коснуться экрана, чтобы он не погас.

🎮 Тест без GPS — симулятор:
https://iliawagen-blip.github.io/moto-hud/sim.html
```

### Пост 5 — Функции

```
⚙️ Что умеет Мото ИЛС

• HUD-навигация — крупная стрелка, скорость, расстояние до поворота (читается через визор)
• Прогноз-дорожка — лента ближайших поворотов (в городе — близко, на трассе — дальше)
• Камеры «в спину» — предупреждение о камерах, которые снимают сзади
• Голос в гарнитуру — манёвры, камеры, приближение к повороту
• Заправки по среднему расходу
• Многодневные поездки с ночёвками
• Импорт из Яндекс.Карт — ссылка или «Поделиться»
• Двойной тап — HUD ↔ карта

Попробовать:
https://iliawagen-blip.github.io/moto-hud/

Симулятор:
https://iliawagen-blip.github.io/moto-hud/sim.html
```
