# Контент-план VK «Мото ИЛС» — 4 недели

Цель: после волны постов 1–8 паблик не выглядит заброшенным. Публикация **каждые 2–3 дня** (≈3 поста в неделю) + ответы в комментариях в день выхода.

Тексты запуска: [`vk-public-setup.md`](vk-public-setup.md) §5.  
Картинки/видео: [`vk-visual-assets.md`](vk-visual-assets.md).

**На старте визуал = только sim.** Полевые скрины и видео с руля — когда появятся, подставлять вместо sim-кадров в те же слоты.

---

## Типы регулярных постов

| Код | Тип | Зачем |
|-----|-----|--------|
| A | HUD-скрин из sim (солнце/ночь/дождь-тема, разные скорости) | Цепляет в ленте, напоминает продукт |
| B | Баг → фикс (до/после) | Прозрачность разработки, доверие |
| C | Отзыв тестера (с разрешения) | Социальное доказательство |
| D | Короткое видео 15–30 с | Максимальный стоп-скролл; пока нет полевого — screen record sim |
| E | Опрос | Вовлечение, идеи в бэклог |
| F | Мини-инструкция | Польза без простыни |
| G | Мем / лёгкий мото-юмор | Только если шутка своя и уместна; иначе пропуск |
| H | Набор в бета / статус недели | Поддержание воронки тестеров |

## Шаблоны-заготовки (короткие)

### A — HUD-скрин

```
[Крючок про скорость / визор / манёвр]

[1–2 предложения: что на скрине — скорость, дистанция, тема]

[Честная оговорка одной строкой, если уместно]

Вопрос в комменты + ссылка на sim.
```

### B — Баг → фикс

```
Нашли [что]. На [какой телефон / sim].

Было: …
Стало: …

Если у тебя так же повторяется — модель телефона в комменты.
```

### E — Опрос

Варианты держать бытовыми, не «продуктовыми»:

- дальняк / город / дождь  
- голос важнее экрана / экран важнее / оба  
- камеры «в спину» — must have / пофиг / бесит лишнее

### D — Видео (пока sim)

Голос за кадром или текст на видео: один манёвр, одна мысль. Без интро «всем привет 30 секунд».

---

## Ритм и анти-заброшенность

- Минимум **3 поста в неделю** после запуска.  
- Каждый выход — **хотя бы один ответ** автору в комментариях в тот же вечер.  
- Статус сообщества (§2 в setup) менять 1–2 раза в неделю под актуальную тему.  
- Если неделя пустая по багам — не выдумывать «релиз»; лучше A + E, чем фальшивый B.  
- Мемы (G) — пропускать без чувства вины. Лучше тихий полезный F.

---

## Когда появятся полевые материалы

Подставить в слоты A/D/C:

1. HUD в солнце (визор + блик)  
2. Ночь / дождь  
3. 15–30 с с рулём и голосом в гарнитуре  
4. Скрин «почти пропустил съезд» с реальной развязки  

До этого календарь полностью закрывается sim + процессом разработки.
